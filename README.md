# Tienda

Tienda de camisetas estampadas

## Instalar dependencias

```bash
npm install
```

## Lanzar aplicación

```bash
# production
npm start

# Copiar .env.production a .env y modificar la variable VITE_APP_URL
cp .env.production .env

# development
npm run dev
```
